## 🚀 使用方法
1. 进入分类目录，点击 `.conf` 文件获取 Raw URL
2. 在 OpenClash 的 配置订阅 中添加该 URL
3. 设置环境变量 `EN_KEY1`, `EN_KEY2` 等为你的订阅链接
